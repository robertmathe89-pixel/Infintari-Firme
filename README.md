# Infintari Firme - Site de prezentare

Site de prezentare pentru Avocat Dibosi Sorin, specializat în înființare societăți comerciale în Timișoara.

## Structură

- `index.html` - Pagina principală
- `styles.css` - Stiluri CSS
- `script.js` - Funcționalități JavaScript
- `package.json` - Configurare npm

## Secțiuni

1. Hero - propunere principală + formular rapid
2. Servicii - tipuri de firme și servicii oferite
3. Cum funcționează - 4 pași simpli
4. De ce noi - argumente de vânzare
5. FAQ - întrebări frecvente
6. Contact - formular și date de contact

## Cum se rulează local

```bash
npm start
```

sau

```bash
npx serve .
```

## Deploy

Site static - poate fi deployat pe orice hosting (Vercel, Netlify, Globe Hosting, etc.).
